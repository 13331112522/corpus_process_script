
## Split Chinese Sentence To Char ##

- the script implement split chinese sentence to character.

## Usage ##
	python split_zh_char.py -h
	python split_zh_char.py --input input_file --output output_file

## Requirment ##
	python --- 3.6

## Output ##
	input:
	中国国庆假期香江将涌入人潮
	output:
	中 国 国 庆 假 期 香 江 将 涌 入 人 潮
